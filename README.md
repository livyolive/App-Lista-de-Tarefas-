# App-Lista-de-Tarefas-
Programa de lista de tarefas para obtenção da terceira nota da disciplina Engenharia de Software

Programa para listar tarefas e visualizar seus status, utilizando os padres de design Strategy (para definir uma família de algorítmos e colocar cada um deles em uma classe separada - Adicionar tarefas; Listar tarefas; removedor tarefa; Alterar status - e tornar seus objetos intercambiáveis.) e Singleton guarda todas as tarefas e garante que existe apenas uma lista de tarefas no programa (garante que uma classe tenha apenas uma instância). Os padres foram utilizados logo no início do código (foram destacados respeitavelmente, primeiro o singleton e logo após o strategy)
